# fitness-tracker
Fitness Tracker App built with React, TypeScript, Redux, and React Query


Title: Fitness Tracker App

Description:
A React app to log workouts, track stats, and explore exercises. Uses Redux for state, React Query + Axios for fetching, Formik for forms, Styled Components for UI, and localStorage for persistence.

Tech Stack: React, TypeScript, Redux, React Query, Formik, Styled Components, Axios, React Router
